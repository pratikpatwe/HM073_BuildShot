# Design System ‚Äì Kairos

## Overview
Kairos follows a **static, minimal, and user-friendly design system** focused on clarity, consistency, and cognitive ease.  
The platform is designed to help users manage **time, habits, finances, and mental well-being** in a calm and structured environment.

The design avoids visual clutter and unnecessary motion, enabling users to focus on meaningful daily actions.

### What ‚ÄúKairos‚Äù Represents

## Typography

### Font Family
**SF (San Francisco)** ‚Äì Apple System Font

Kairos uses a **single static font family** across the entire platform to ensure visual consistency and high readability.

#### Font Usage
- **Headings:** SF Pro Display (Semibold / Bold)
- **Body Text:** SF Pro Text (Regular)
- **UI Labels & Metadata:** SF Pro Text (Medium)

No decorative or secondary fonts are used.

---

## Color System

### Primary Theme
Kairos uses a **static Black + Green gradient theme**, representing focus, growth, and balance.

#### Primary Colors
- **Primary Black:** `#0B0B0B`
- **Green Gradient:**  
  - Start: `#00C853`
  - End: `#1DB954`

Used for:
- Primary actions
- Habit progress indicators
- Focus score visuals
- Highlights and accents

### Supporting Colors
- **White:** `#FFFFFF` ‚Äì Primary text
- **Muted Gray:** `#A0A0A0` ‚Äì Secondary text
- **Dark Gray:** `#1A1A1A` ‚Äì Cards and containers

The color palette is intentionally limited to maintain visual clarity.

## Iconography

### Icon Library
**Lucide Icons**

Kairos uses **Lucide** as its primary icon library.

#### Why Lucide?
- Minimal, clean, and modern design
- Consistent stroke-based icons
- Open-source and lightweight
- Perfect fit for dark, static UI themes

#### Icon Usage Guidelines
- Line icons only (no filled or decorative icons)
- Icons used to enhance clarity, not decoration
- Consistent size and stroke width across the platform
- Icons paired with text labels where clarity is required

## Summary
Kairos adopts a **static, minimal, and cognitively friendly design system** to help users manage their life intentionally.  
The black and green gradient theme reinforces focus and growth, while the SF font and Lucide icons ensure clarity, consistency, and a modern interface.

This design approach supports scalability while keeping the MVP simple, usable, and impactful.
